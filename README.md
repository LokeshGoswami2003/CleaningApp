# CleaningApp

Backend-first cleaning-service marketplace similar to Urban Company.

This repo contains:

- `server/`: Express.js + TypeScript API.
- `client-web/`: React + Vite admin/operator client.
- `docs/`: architecture, roadmap, decisions, journey logs, and database flow.

## Start Here

Read:

- `AGENTS.md`
- `docs/PLAN.md`
- `docs/setup/phase-1-setup.md`
- `docs/development/commands.md`

## Phase 1 Goal

Create the foundation:

- Express API skeleton.
- Health endpoint.
- Standard response/error format.
- PostgreSQL + Prisma setup.
- Local Docker database flow.
- React admin folder structure.
- Agent and documentation guardrails.
